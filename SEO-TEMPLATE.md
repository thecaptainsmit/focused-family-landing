# SEO Template for Blog Posts

## Pre-Writing Checklist

- [ ] Target keyword identified (use Google Trends, low competition)
- [ ] Search intent matched (informational vs transactional)
- [ ] Related keywords identified (3-5 LSI keywords)
- [ ] Internal linking opportunities mapped

## On-Page SEO Requirements

### Title Tag
- 50-60 characters max
- Include target keyword near beginning
- Compelling click-worthy headline
- Format: "Primary Keyword | Secondary Keyword - Brand"
- Example: "ADHD Morning Routine Chart That Works | Visual Schedule for Kids"

### Meta Description
- 150-160 characters max
- Include target keyword
- Include call-to-action
- Compelling summary of content
- Example: "Learn why most ADHD morning routine charts fail and discover the science-backed system that reduces morning battles by 64%."

### URL Structure
- Short, descriptive, keyword-rich
- Use hyphens (not underscores)
- No stop words (a, an, the, in, etc.)
- Example: `/blog/adhd-morning-routine-chart` not `/blog/why-most-morning-routine-charts-fail`

### Header Structure
```
H1: Main title (only one per page)
  H2: Major sections
    H3: Subsections
    H3: Subsections
  H2: Major sections
    H3: Subsections
```

### Content Guidelines
- [ ] Target keyword in first 100 words
- [ ] Target keyword in H1
- [ ] Target keyword in at least one H2
- [ ] Related keywords naturally throughout
- [ ] Content length: 1,500-2,500 words (comprehensive)
- [ ] Short paragraphs (2-3 sentences max)
- [ ] Use bullet points and numbered lists
- [ ] Include images with alt text
- [ ] Include statistics and data (with sources)
- [ ] Include expert quotes or testimonials

### Internal Linking
- [ ] 3-5 internal links to related posts
- [ ] 1-2 internal links to product/service pages
- [ ] Descriptive anchor text (not "click here")
- [ ] Link to cornerstone content

### External Linking
- [ ] 2-3 external links to authoritative sources
- [ ] CDC, medical journals, research papers
- [ ] Open in new tab (target="_blank" rel="noopener")

### Schema Markup
Include JSON-LD structured data:
- BlogPosting schema
- Author information
- Publication date
- Modified date (if updated)

### Technical SEO
- [ ] Mobile responsive design
- [ ] Fast page load (<3 seconds)
- [ ] SSL certificate (HTTPS)
- [ ] XML sitemap updated
- [ ] Robots.txt allows indexing

### Social Meta Tags
- [ ] Open Graph title
- [ ] Open Graph description
- [ ] Open Graph image (1200x630px)
- [ ] Twitter Card meta tags

## Content Quality Checklist

- [ ] Solves a specific problem
- [ ] Evidence-based claims (cite sources)
- [ ] Actionable takeaways
- [ ] Unique insights (not just rehashed content)
- [ ] Easy to scan (headers, bold, lists)
- [ ] Engaging intro that hooks
- [ ] Strong conclusion with CTA

## Post-Publishing Checklist

- [ ] Submit to Google Search Console
- [ ] Share on social media
- [ ] Email to list
- [ ] Monitor rankings (week 1, 2, 4)
- [ ] Update content every 6-12 months

## Link Building Strategy

### Internal Linking Map
- Every post links to 3-5 related posts
- Every post links to cornerstone content
- Product pages linked contextually
- Create content silos by topic

### External Linking Strategy
- Link to CDC, NIH, CHADD
- Link to research papers
- Link to ADHD experts
- Link to book references

## Conversion Optimization

### Lead Magnets
- Include relevant free resource
- Contextual CTA (not just at end)
- Email capture form
- Content upgrade (PDF version)

### Product CTAs
- Soft sell in middle of content
- Hard sell at end
- Use benefit-focused language
- Include social proof

## Analytics Tracking

### KPIs to Monitor
- Organic search traffic
- Keyword rankings
- Click-through rate (CTR)
- Bounce rate
- Time on page
- Conversion rate
- Email signups

### Tools
- Google Analytics 4
- Google Search Console
- Ahrefs or SEMrush (if available)
- Heatmaps (Hotjar)
